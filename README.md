# Rua Viva

Plataforma desenvolvida como Trabalho de Conclusão de Curso do curso de Sistemas de Informação.

## Sobre o projeto

A **plataforma Rua Viva** foi desenvolvida com o objetivo de apoiar o registro, mapeamento e acompanhamento de pessoas em situação de vulnerabilidade social, promovendo a conexão entre usuários, parceiros e informações georreferenciadas.

O projeto foi construído em **Flutter**, com persistência local utilizando **Hive**, e contempla funcionalidades de cadastro de ocorrências, cadastro de parceiros, visualização em mapa e compartilhamento de informações.

## Objetivo

Oferecer uma plataforma tecnológica de apoio social, capaz de auxiliar no registro e organização de informações relevantes para atendimento de pessoas em situação de vulnerabilidade, contribuindo para ações mais estruturadas e acessíveis.

## Tecnologias utilizadas

- Flutter
- Dart
- Hive
- Google Maps
- Android / Web / Desktop

## Principais funcionalidades

- Cadastro de ocorrências
- Cadastro de parceiros
- Listagem de registros
- Visualização em mapa
- Compartilhamento de informações
- Organização de dados para apoio social
